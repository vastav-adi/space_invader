# space_invader
A simple space_invader game made using pygame.

	Requirements:
	>	python 3.x
	> pygame
	
	How to run this game:
	> clone or download the repo.
	> pip install pygame
	> open terminal within the folder
	> python main.py
	

p.s: there is no end to the game. It is not a bug, it is a feature <3
